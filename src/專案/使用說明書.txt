注意事項:
    1.需要安裝的東西，有可能會更多或更少
    2.需要以系統管理員身分下(SQLite功能)的cmd(密碼屏蔽功能)開啟

請安裝:
    pip install requests
    pip install json
    pip install theading
    pip install selenium
    pip install beautifulsoup4
    pip install smtplib

    1.以python開發
    2.軟工.py是整個專案的主畫面(開始畫面、主程式)
    3.sub目錄下，放各個功能的code(副程式)
